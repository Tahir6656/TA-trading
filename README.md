# TA Trading - Technical Analysis Trading Signal Generator

A web-based application that analyzes financial charts using technical indicators and provides trade signal suggestions with risk management tools.

## 🎯 Features

### Technical Indicators
- RSI (Relative Strength Index)
- MACD (Moving Average Convergence Divergence)
- Moving Averages (SMA & EMA)
- Bollinger Bands

### Risk Management
- Position sizing calculator
- Stop-loss recommendations
- Take-profit level suggestions
- Risk-reward ratio calculator

### Dashboard
- Real-time chart visualization
- Technical indicator overlays
- Trade signal suggestions (UP/DOWN/HOLD)

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB
- Charts: TradingView Lightweight Charts

---

## 🚀 Quick Start

Backend:
```bash
cd backend
npm install
npm run dev
```

Frontend:
```bash
cd frontend
npm install
npm start
```
